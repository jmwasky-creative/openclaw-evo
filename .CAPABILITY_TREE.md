## 🎯 能力使用原则

### 面对新任务时
1. **定位**: 在能力树中查找匹配节点
2. **组合**: 组合现有能力节点
3. **评估**: 判断是否需要生长新节点
4. **执行**: 沿能力树路径执行

### 能力生长规则
1. **挂载**: 新能力必须挂载到已有节点
2. **定义**: 必须明确输入/输出/前提/失败
3. **验证**: 必须可测试、可复现
4. **文档**: 更新能力树

### 能力修剪规则
1. **标记**: 长期未触发、低收益能力
2. **合并**: 相似能力合并为通用节点
3. **存档**: 保留到历史记录，不删除
4. **清理**: 仅在确认无用后移除
